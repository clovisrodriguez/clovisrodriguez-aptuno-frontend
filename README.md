## Correr el proyecto

Es super sencillo solo tienen que hacer un:

```
yarn install
```

Seguido de un

```
yarn start
```

Rodar Pruebas

```
yarn test
```

Cubrimiento de testing del base code

```
yarn testCoverage

```

Pruebas con hot realoding

```
yarn testWatch

```
